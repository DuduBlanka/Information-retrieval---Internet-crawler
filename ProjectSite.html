Queries Run Time

How to improve the queries run time?

There are several factors that can improve the run time:

  • Connectivity: In order to retrieve information from the website, our code opens a chrome page, enters the query we wish to run on the site, and then proceeds by visiting each product returned by the query, so the more faster the internet connection, the better the run time would be.

  • Stability: In our project we ran into stability issues with the website, where it would cause the chrome page to freeze until the site goes back online, so the more stable the site itself is, the better the run time would be.
